# Introducción

No todo sigue una Normal o Exponencial. Dependiendo del fenómeno, necesitamos otras distribuciones:

| Distribución | Uso Principal |
|--------------|---------------|
| **Uniforme** | Números aleatorios, loterías |
| **Gamma** | Tiempo hasta k eventos, tiempos de servicio |
| **Weibull** | Análisis de fiabilidad con envejecimiento |
| **Beta** | Proporciones, probabilidades bayesianas |

---

# 1. Distribución Uniforme Continua

La más simple: todos los valores en un intervalo $[a, b]$ son igualmente probables.

### Parámetros
- $a$: Límite inferior
- $b$: Límite superior

### Fórmulas
$$f(x) = \frac{1}{b-a}, \quad a \leq x \leq b$$

$$E[X] = \frac{a+b}{2}, \quad Var(X) = \frac{(b-a)^2}{12}$$

### Ejemplo: Generador de Números Aleatorios
`random.random()` genera números uniformes en $[0, 1]$.

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import uniform

# Uniforme entre 10 y 20
a, b = 10, 20
loc = a            # SciPy usa 'loc' como el punto de inicio
scale = b - a      # 'scale' es el ancho del intervalo

# Generar 1000 muestras
muestras = uniform.rvs(loc=loc, scale=scale, size=1000)

plt.figure(figsize=(10, 5))
plt.hist(muestras, bins=20, density=True, alpha=0.7, color='skyblue', edgecolor='black')
plt.axhline(1/scale, color='red', linestyle='--', label=f'Densidad = 1/{scale} = {1/scale:.2f}')
plt.title(f'Uniforme({a}, {b}) - 1000 muestras')
plt.xlabel('Valor')
plt.ylabel('Densidad')
plt.legend()
plt.show()

print(f"Media teórica: {(a+b)/2}, Media muestral: {np.mean(muestras):.2f}")
```

# 2. Distribución Gamma

Generaliza la exponencial para modelar el tiempo hasta **k eventos** (no solo 1).

### Parámetros
- $k$ (shape): Número de eventos a esperar (α en SciPy)
- $\theta$ (scale): Tiempo medio por evento (1/λ)

### Relación con Exponencial
$$\text{Gamma}(k=1, \theta) = \text{Exponencial}(\lambda = 1/\theta)$$

### Uso Típico
- Tiempo total de k llamadas telefónicas
- Suma de tiempos de servicio

```python
from scipy.stats import gamma
import matplotlib.pyplot as plt
import numpy as np

# Tiempo hasta 3 llamadas (k=3), con media de 5 min/llamada
k = 3
theta = 5  # scale

x = np.linspace(0, 40, 200)
y = gamma.pdf(x, a=k, scale=theta)

plt.figure(figsize=(10, 5))
plt.plot(x, y, 'b-', linewidth=2, label=f'Gamma(k={k}, θ={theta})')
plt.axvline(k * theta, color='green', linestyle='--', label=f'Media = k×θ = {k*theta}')
plt.title('Distribución Gamma: Tiempo hasta 3 llamadas')
plt.xlabel('Tiempo (minutos)')
plt.ylabel('Densidad')
plt.legend()
plt.show()
```

# 3. Cuándo Usar Cada Distribución

| Pregunta | Distribución |
|----------|--------------|
| ¿Tiempo hasta el primer evento? | Exponencial |
| ¿Tiempo hasta el k-ésimo evento? | Gamma |
| ¿Valor aleatorio uniforme en [a,b]? | Uniforme |
| ¿Fallas con tasa variable por edad? | Weibull |
| ¿Proporción desconocida (Bayesiano)? | Beta |

<Callout type="info">
**Tip:** En la vida real, primero grafica tus datos (histograma) y luego decide qué distribución "encaja" mejor.
</Callout>
