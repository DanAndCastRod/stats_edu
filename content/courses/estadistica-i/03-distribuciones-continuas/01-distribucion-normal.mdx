# Introducción

> **Hecho:** Tu estatura, tu CI, los errores de manufactura, el rendimiento en un examen... casi todo en la naturaleza sigue una **Distribución Normal**.

La Distribución Normal (o Gaussiana) es tan omnipresente que se le llama "la ley de los errores". Fue descubierta por Carl Friedrich Gauss mientras estudiaba errores en mediciones astronómicas.

<Callout type="info">
**¿Por qué importa?** Casi todas las técnicas de inferencia estadística (intervalos de confianza, pruebas de hipótesis) asumen que los datos son normales o se aproximan a la normal.
</Callout>

## Objetivos de Aprendizaje

# 1. Anatomía de la Curva Normal

La distribución normal es **simétrica** y tiene forma de **campana**.

### Parámetros
- **$\mu$ (mu):** La media. Define dónde está el centro.
- **$\sigma$ (sigma):** La desviación estándar. Define qué tan "ancha" es la campana.

### La Regla Empírica (68-95-99.7)
- **68%** de los datos están entre $\mu \pm 1\sigma$
- **95%** de los datos están entre $\mu \pm 2\sigma$
- **99.7%** de los datos están entre $\mu \pm 3\sigma$

# 2. Estandarización: El Puntaje Z

Para comparar datos de diferentes escalas, los convertimos a una **Normal Estándar** ($\mu=0$, $\sigma=1$).

$$Z = \frac{X - \mu}{\sigma}$$

### Ejemplo
Un estudiante saca 85 en un examen donde la media es 70 y la desviación 10.
$$Z = \frac{85 - 70}{10} = 1.5$$

Esto significa que el estudiante está **1.5 desviaciones estándar por encima de la media**. ¡Excelente!

# 3. Calculando Probabilidades

En lugar de usar tablas Z (método anticuado), usaremos Python.

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

# Parámetros de nuestra distribución
mu = 70      # Media del examen
sigma = 10   # Desviación estándar

# Pregunta: ¿Qué probabilidad hay de sacar más de 85?
prob_mayor_85 = 1 - norm.cdf(85, mu, sigma)
print(f"P(X > 85) = {prob_mayor_85:.4f} = {prob_mayor_85*100:.2f}%")

# Pregunta: ¿Entre qué valores está el 95% central?
limite_inferior = norm.ppf(0.025, mu, sigma)
limite_superior = norm.ppf(0.975, mu, sigma)
print(f"El 95% de los estudiantes sacan entre {limite_inferior:.1f} y {limite_superior:.1f}")

# Visualización
x = np.linspace(mu - 4*sigma, mu + 4*sigma, 200)
y = norm.pdf(x, mu, sigma)

plt.figure(figsize=(10, 6))
plt.plot(x, y, 'b-', linewidth=2, label='Distribución Normal')
plt.fill_between(x, y, where=(x >= 85), alpha=0.3, color='red', label=f'P(X>85) = {prob_mayor_85:.2%}')
plt.axvline(mu, color='green', linestyle='--', label=f'μ = {mu}')
plt.title(f'Distribución Normal (μ={mu}, σ={sigma})')
plt.xlabel('Calificación')
plt.ylabel('Densidad de Probabilidad')
plt.legend()
plt.show()
```

# 4. Funciones Clave de `scipy.stats.norm`

| Función | Descripción | Uso |
|---------|-------------|-----|
| `norm.pdf(x, μ, σ)` | Densidad en x | Altura de la curva |
| `norm.cdf(x, μ, σ)` | P(X ≤ x) | Área a la izquierda |
| `norm.ppf(q, μ, σ)` | Cuantil q | Valor de X dado un % |
| `norm.rvs(μ, σ, size)` | Generar datos | Simulación |

<Callout type="warning">
**Cuidado:** `norm.ppf(0.95)` te da el valor donde el 95% está **a la izquierda**. Si necesitas el 95% central, usa `norm.ppf(0.025)` y `norm.ppf(0.975)`.
</Callout>
