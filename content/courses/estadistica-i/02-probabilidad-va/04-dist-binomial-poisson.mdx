# Introducción

No necesitamos reinventar la rueda. Muchos fenómenos naturales siguen patrones predecibles.
Las **Distribuciones de Probabilidad** son modelos matemáticos que "ajustan" a estos fenómenos.

- ¿Lanzar una moneda 10 veces? -> **Binomial**.
- ¿Número de clientes llegando al banco por hora? -> **Poisson**.

---

# 1. Distribución Binomial ($n, p$)

Modela el número de "éxitos" en $n$ intentos independientes, cada uno con probabilidad $p$ de éxito.

$$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$$

### Parámetros
- $n$: Número de ensayos (e.g., 100 piezas inspeccionadas).
- $p$: Probabilidad de defecto (e.g., 0.05).

# 2. Distribución Poisson ($\lambda$)

Modela el número de eventos que ocurren en un intervalo de tiempo o espacio fijo.

$$P(X=k) = \frac{e^{-\lambda} \lambda^k}{k!}$$

### Parámetro
- $\lambda$ (Lambda): Tasa promedio de ocurrencia (e.g., 5 llamadas/minuto).

# Lab: Visualizando las Distribuciones

Usemos `scipy.stats` para calcular probabilidades exactas y visualizar las distribuciones.

```python
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy.stats import binom, poisson

# Configuración
sns.set_theme(style="whitegrid")
plt.figure(figsize=(12, 5))

# --- BINOMIAL ---
# Ejemplo: 20 lanzamientos de moneda (p=0.5)
n, p = 20, 0.5
x_binom = np.arange(0, n+1)
pmf_binom = binom.pmf(x_binom, n, p)

plt.subplot(1, 2, 1)
plt.bar(x_binom, pmf_binom, color='skyblue', label=f'Binomial(n={n}, p={p})')
plt.title(f'Distribución Binomial (n={n}, p={p})')
plt.xlabel('Número de Éxitos')
plt.ylabel('Probabilidad')
plt.legend()

# --- POISSON ---
# Ejemplo: Promedio de 5 llamadas por minuto
mu = 5
x_poisson = np.arange(0, 15)
pmf_poisson = poisson.pmf(x_poisson, mu)

plt.subplot(1, 2, 2)
plt.bar(x_poisson, pmf_poisson, color='salmon', label=f'Poisson(λ={mu})')
plt.title(f'Distribución Poisson (λ={mu})')
plt.xlabel('Número de Eventos')
plt.legend()

plt.tight_layout()
plt.show()

# Cálculos Exactos
print(f"Probabilidad de obtener exactamente 10 caras (Binomial): {binom.pmf(10, n, p):.4f}")
print(f"Probabilidad de recibir exactamente 5 llamadas (Poisson): {poisson.pmf(5, mu):.4f}")
```

<Callout type="info">
**Tip de Ingeniería:** Cuando $n$ es muy grande y $p$ es muy pequeño (eventos raros), la Binomial se parece mucho a la Poisson.
</Callout>
