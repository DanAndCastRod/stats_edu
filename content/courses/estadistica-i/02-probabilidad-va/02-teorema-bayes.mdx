# Introducción

> **Dilema:** Te haces una prueba de una enfermedad rara (afecta al 1% de la población). La prueba tiene 99% de precisión. Sale positiva. ¿Estás enfermo?

Tu intuición grita "¡Sí, al 99%!".
El **Teorema de Bayes** te dice la verdad: Tu probabilidad real es solo **50%**.

¿Por qué? Porque ignoraste la "Probabilidad A Priori" (la enfermedad es muy rara). En esta lección aprenderemos a actualizar nuestras probabilidades basándonos en nueva evidencia.

<Callout type="info">
**Aplicación Real:** Filtros de Spam, Diagnósticos Médicos, Machine Learning (Clasificadores Naive Bayes).
</Callout>

# 1. La Fórmula Maestra

Bayes relaciona dos eventos condicionales opuestos: $P(A|B)$ y $P(B|A)$.

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

Donde:
- **$P(A)$**: Probabilidad A Priori (Creencia inicial).
- **$P(B|A)$**: Verosimilitud (Probabilidad de ver la evidencia $B$ si $A$ es verdad).
- **$P(B)$**: Probabilidad Total de la Evidencia (Normalizador).
- **$P(A|B)$**: Probabilidad A Posteriori (Creencia actualizada).

# 2. El Ejemplo Médico (Paso a Paso)

Vamos a resolver el dilema inicial.
- **Enfermedad ($E$):** 1% de la población ($P(E) = 0.01$).
- **Salud ($S$):** 99% de la población ($P(S) = 0.99$).
- **Prueba (+):**
    - Sensibilidad: $P(+|E) = 0.99$ (Detecta el 99% de enfermos).
    - Falsos Positivos: $P(+|S) = 0.01$ (Se equivoca en 1% de sanos).

Queremos saber: **$P(E|+)$** (Probabilidad de estar enfermo dado que dio positivo).

### Cálculo

1.  **Numerador (Verdaderos Positivos):**
    $$P(+|E) \cdot P(E) = 0.99 \cdot 0.01 = 0.0099$$

2.  **Denominador (Total de Positivos):**
    Salen positivos los enfermos reales + los sanos falsamente alarmados.
    $$P(+) = (0.99 \cdot 0.01) + (0.01 \cdot 0.99) = 0.0099 + 0.0099 = 0.0198$$

3.  **Resultado (Bayes):**
    $$P(E|+) = \frac{0.0099}{0.0198} = 0.5$$

**Conclusión:** ¡Solo tienes un 50% de probabilidad! Hay tantos "falsos positivos" generados por la gran cantidad de gente sana, que igualan a los verdaderos positivos.

# Lab: Calculadora Bayesiana

Usa este script para simular diferentes escenarios. ¿Qué pasa si la enfermedad es más común (10%)? ¿Qué pasa si la prueba es perfecta (100%)?

```python
def teorema_bayes(p_priori, sensibilidad, especificidad):
    """
    Calcula P(Enfermo|Positivo)
    p_priori: Tasa base de la enfermedad (0-1)
    sensibilidad: Probabilidad de + dado Enfermo
    especificidad: Probabilidad de - dado Sano
    """
    p_sano = 1 - p_priori
    p_falso_positivo = 1 - especificidad
    
    # Numerador: Verdaderos Positivos
    verdaderos_positivos = sensibilidad * p_priori
    
    # Denominador: Total Positivos (Verdaderos + Falsos)
    falsos_positivos = p_falso_positivo * p_sano
    total_positivos = verdaderos_positivos + falsos_positivos
    
    if total_positivos == 0: return 0
    
    p_posteriori = verdaderos_positivos / total_positivos
    return p_posteriori

# Escenario 1: Enfermedad Rara
res1 = teorema_bayes(p_priori=0.01, sensibilidad=0.99, especificidad=0.99)
print(f"Escenario 1 (Rara): {res1:.2%}")

# Escenario 2: Pandemia (Enfermedad Común = 20%)
res2 = teorema_bayes(p_priori=0.20, sensibilidad=0.90, especificidad=0.90)
print(f"Escenario 2 (Pandemia): {res2:.2%}")

# Escenario 3: Prueba Mala (Tirar una moneda)
res3 = teorema_bayes(p_priori=0.01, sensibilidad=0.50, especificidad=0.50)
print(f"Escenario 3 (Moneda): {res3:.2%}")
```

<Callout type="warning">
**Intuición:** Bayes nos enseña que **afirmaciones extraordinarias requieren evidencia extraordinaria**. Si algo es muy raro a priori (un milagro, una enfermedad exótica), una evidencia "bastante buena" a menudo no es suficiente para confirmar su veracidad.
</Callout>
