# Introducción

El mundo está lleno de eventos cualitativos: "Llovió", "Salió cara", "El cliente compró".
Para hacer ingeniería, necesitamos **números**.
Una **Variable Aleatoria (X)** es una función que traduce estos eventos a números reales.

- Evento: "Salió Cara" -> $X = 1$
- Evento: "Salió Sello" -> $X = 0$

<Callout type="info">
**Concepto Clave:** "Aleatoria" no significa "Cualquier cosa". Significa que el resultado depende del azar, pero sus *valores posibles* y sus *probabilidades* están bien definidos.
</Callout>

## Objetivos de Aprendizaje
- Diferenciar VA Discretas (saltos) vs Continuas (fluido).
- Calcular e interpretar el **Valor Esperado** $E[X]$.
- Graficar la Función de Masa de Probabilidad (PMF).

# 1. Función de Masa de Probabilidad (PMF)

Para una variable discreta, la PMF nos dice la probabilidad exacta de cada valor.
$$P(X = x)$$

### Ejemplo: Lanzar dos dados
Sea $X$ la suma de las caras.
- Mínimo: $1+1=2$
- Máximo: $6+6=12$
- ¿Cuál es más probable? El 7 (tienes 6 formas de lograrlo: 1+6, 2+5, 3+4...).

<Callout type="warning">
**La intuición falla:** A veces, las probabilidades no son lo que parecen.
</Callout>

<Callout type="note">
**Paradoja de Monty Hall:** En este clásico problema, tu mejor estrategia es cambiar de puerta. La probabilidad sube de 1/3 a 2/3. ¡Juega y compruébalo!
</Callout>

# 2. Valor Esperado ($E[X]$)

Es el "centro de gravedad" de la distribución. Es lo que esperarías obtener en promedio si repitieras el experimento infinitas veces. **No es necesariamente un valor posible**.

$$E[X] = \sum x \cdot P(x)$$

### Ejemplo: El Juego de la Feria
- Ganas \$100 con prob 0.1
- Pierdes \$5 con prob 0.9

$$E[X] = (100 \cdot 0.1) + (-5 \cdot 0.9) = 10 - 4.5 = \$5.5$$
¡Es un juego rentable! (A largo plazo).

# Lab: Simulando el Valor Esperado

Vamos a simular el lanzamiento de dados miles de veces para ver cómo el promedio empírico converge al Valor Esperado teórico.

```python
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# 1. Definir la VA (Suma de 2 dados)
# Espacio muestral teórico
d1 = np.arange(1, 7)
d2 = np.arange(1, 7)
teorico = [x+y for x in d1 for y in d2]

valor_esperado_teorico = np.mean(teorico)
print(f"Valor Esperado Teórico (Centro de Gravedad): {valor_esperado_teorico}")

# 2. Simulación Monte Carlo
n_simulaciones = 1000
resultados = []

# Lanzamos los dados n veces
sim_d1 = np.random.randint(1, 7, n_simulaciones)
sim_d2 = np.random.randint(1, 7, n_simulaciones)
resultados = sim_d1 + sim_d2

promedio_empirico = np.mean(resultados)
print(f"Promedio Empírico ({n_simulaciones} intentos): {promedio_empirico:.4f}")

# 3. Gráfica
plt.figure(figsize=(10, 6))
sns.histplot(resultados, bins=11, stat='probability', discrete=True, color='purple', alpha=0.6, label='Simulación')
plt.axvline(valor_esperado_teorico, color='red', linestyle='--', label=f'E[X]={valor_esperado_teorico}')
plt.title('Convergecia al Valor Esperado (Ley de los Grandes Números)')
plt.legend()
plt.show()
```

<Callout type="warning">
**Varianza ($Var(X)$):** Mide qué tan dispersos están los valores respecto al centro. En finanzas, esto es sinónimo de **Riesgo**.
</Callout>
