La respuesta ingenua es "muchos". La estadística nos da la respuesta exacta: $10^4 = 10,000$ intentos.

# 1. El Principio Multiplicativo

Es la regla de oro: Si tienes $n$ formas de hacer una cosa y $m$ formas de hacer otra, tienes $n \times m$ formas de hacer ambas.

### Ejemplo: El Menú de la Cafetería
- **Plato Fuerte:** Hamburguesa, Perro, Pizza (3 opciones)
- **Bebida:** Gaseosa, Jugo (2 opciones)
- **Postre:** Helado, Galleta (2 opciones)

Total de combos = $3 \times 2 \times 2 = 12$ combinaciones diferentes.

```python
# Verificación en Python
platos = ['Hamburguesa', 'Perro', 'Pizza']
bebidas = ['Gaseosa', 'Jugo']
postres = ['Helado', 'Galleta']

count = 0
for p in platos:
    for b in bebidas:
        for d in postres:
            count += 1
            print(f"{count}: {p} + {b} + {d}")
            
print(f"Total calculado: {count}")
```

# 2. Factorial ($n!$)

El factorial de un número es la multiplicación de todos los enteros decrecientes hasta 1. Representa la cantidad de formas de **ordenar** $n$ objetos distintos.

$$n! = n \times (n-1) \times (n-2) \times ... \times 1$$

- $3! = 3 \times 2 \times 1 = 6$
- $5! = 120$
- **Ojo:** $0! = 1$ (Por definición matemática).

```python
import math

n = 5
print(f"Factorial de {n} es: {math.factorial(n)}")
```

# 3. Permutaciones (¡El Orden Importa!)

Usamos permutaciones cuando el orden de los elementos altera el resultado.
*Ejemplo: Una carrera de F1. No es lo mismo llegar de 1ro que de 2do.*

**Fórmula:** Si queremos ordenar $r$ objetos de un grupo de $n$:

$$P(n, r) = \frac{n!}{(n-r)!}$$

### Ejemplo: El Podium
Tenemos 10 corredores. ¿De cuántas formas podemos llenar el podio (Oro, Plata, Bronce)?
$n=10$, $r=3$

$$P(10, 3) = \frac{10!}{7!} = 10 \times 9 \times 8 = 720 \text{ formas}$$

---

# 4. Combinaciones (¡El Orden NO Importa!)

Usamos combinaciones cuando el orden es irrelevante. Solo importa "quiénes están".
*Ejemplo: Un comité de estudiantes. Elegir a Ana y Juan es lo mismo que elegir a Juan y Ana.*

**Fórmula:** Dividimos las permutaciones por $r!$ para eliminar los duplicados por orden.

$$C(n, r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}$$

### Ejemplo: La Lotería (Baloto)
Debes elegir 5 números de 45 opciones. No importa en qué orden salgan las balotas.

$$C(45, 5) = \frac{45!}{5! \times 40!} = \text{Un número muy grande}$$

```python
import math

# Función personalizada para combinaciones
def combinatoria(n, r):
    return math.factorial(n) // (math.factorial(r) * math.factorial(n - r))

posibilidades = combinatoria(45, 5)
print(f"Posibilidades de ganar el Baloto (solo 5 números): {posibilidades:,}")
print(f"Probabilidad de ganar: 1 en {posibilidades:,}")
```

# Lab: Itertools en Acción

Python tiene una librería estándar llamada `itertools` que hace todo el trabajo sucio por nosotros.

```python
import itertools

letras = ['A', 'B', 'C', 'D']

# 1. Permutaciones de 2 letras (Importa orden: AB != BA)
print("--- Permutaciones (2) ---")
perms = list(itertools.permutations(letras, 2))
print(f"Cantidad: {len(perms)}")
print(perms)

# 2. Combinaciones de 2 letras (No importa orden: AB == BA, solo sale AB)
print("\n--- Combinaciones (2) ---")
combs = list(itertools.combinations(letras, 2))
print(f"Cantidad: {len(combs)}")
print(combs)
```

<Callout type="warning">
**Cuidado Computacional:** No intentes calcular `permutations` de listas grandes (e.g., 20 elementos). El crecimiento factorial desbordará la memoria de tu navegador rápidamente.
</Callout>
