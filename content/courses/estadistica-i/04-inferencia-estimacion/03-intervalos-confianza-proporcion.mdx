# Introducción

> **Escenario:** Una encuesta a 400 personas muestra que 180 prefieren el producto A. ¿Cuál es el verdadero % de preferencia en toda la población?

Cuando el parámetro de interés es una **proporción** (p), usamos un IC diferente.

<Callout type="info">
**Aplicaciones:** Tasas de defecto en manufactura, porcentaje de votos, tasa de conversión en marketing, prevalencia de enfermedades.
</Callout>

## Objetivos de Aprendizaje
- Calcular ICs para proporciones usando la aproximación Normal.
- Verificar las condiciones de validez (np ≥ 10, n(1-p) ≥ 10).
- Interpretar resultados en contexto.

---

# 1. Fórmula del IC para Proporciones

$$\hat{p} \pm Z_{\alpha/2} \cdot \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}$$

Donde:
- $\hat{p} = \frac{x}{n}$ = proporción muestral
- $x$ = número de "éxitos"
- $n$ = tamaño de muestra

### Condiciones de Validez
Para que la aproximación Normal funcione:
- $n\hat{p} \geq 10$
- $n(1-\hat{p}) \geq 10$

# 2. Ejemplo Paso a Paso

**Problema:** De 500 piezas inspeccionadas, 35 tienen defectos. Construir IC 95% para la tasa de defectos.

```python
from scipy.stats import norm
import numpy as np

# Datos
x = 35              # Defectuosas
n = 500             # Total inspeccionadas
confianza = 0.95

# Cálculos
p_hat = x / n
alpha = 1 - confianza
z_critico = norm.ppf(1 - alpha/2)

# Error estándar de la proporción
ee = np.sqrt(p_hat * (1 - p_hat) / n)
margen_error = z_critico * ee

ic_inferior = p_hat - margen_error
ic_superior = p_hat + margen_error

# Verificar condiciones
print("--- Verificación ---")
print(f"n·p̂ = {n * p_hat:.1f} (debe ser ≥ 10) ✓")
print(f"n·(1-p̂) = {n * (1-p_hat):.1f} (debe ser ≥ 10) ✓")

print("\n--- Resultados ---")
print(f"Proporción muestral: {p_hat:.4f} = {p_hat*100:.2f}%")
print(f"IC {confianza*100:.0f}%: ({ic_inferior:.4f}, {ic_superior:.4f})")
print(f"En porcentaje: ({ic_inferior*100:.2f}%, {ic_superior*100:.2f}%)")
```

---

# 3. Interpretación en Contexto

Del ejemplo anterior:
- Tasa de defectos estimada: **7%**
- IC 95%: **(4.8%, 9.2%)**

**Interpretación:** Con 95% de confianza, la verdadera tasa de defectos de la línea de producción está entre 4.8% y 9.2%.

<Callout type="warning">
**Decisión Gerencial:** Si el estándar de calidad exige menos de 5% de defectos, este IC sugiere que probablemente NO se cumple (el límite inferior es 4.8%, muy cerca).
</Callout>

# 4. IC de Agresti-Coull (Ajustado)

Para muestras pequeñas o proporciones extremas, el método clásico puede fallar. El método Agresti-Coull agrega 2 "éxitos" y 2 "fracasos" ficticios.

$$\tilde{p} = \frac{x + 2}{n + 4}$$

```python
# Método Agresti-Coull
x_adj = x + 2
n_adj = n + 4
p_tilde = x_adj / n_adj

ee_adj = np.sqrt(p_tilde * (1 - p_tilde) / n_adj)
margen_adj = z_critico * ee_adj

ic_inf_adj = p_tilde - margen_adj
ic_sup_adj = p_tilde + margen_adj

print(f"IC Agresti-Coull: ({ic_inf_adj*100:.2f}%, {ic_sup_adj*100:.2f}%)")
```
