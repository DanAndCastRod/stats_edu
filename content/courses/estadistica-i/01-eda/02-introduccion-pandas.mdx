## 1. El Objeto DataFrame

La estructura central de Pandas es el `DataFrame`. Es una tabla bidimensional con:
- **Índice (Index):** Las etiquetas de las filas (eje 0).
- **Columnas (Columns):** Las etiquetas de las variables (eje 1).

```python
import pandas as pd
import io

# Simulamos un archivo CSV en memoria para este ejemplo
csv_content = """id,maquina,temperatura,falla
1,M1,120.5,False
2,M2,110.2,False
3,M1,125.0,True
4,M3,115.8,False"""

# Creamos el archivo 'produccion.csv' en el sistema de archivos virtual
with open('produccion.csv', 'w') as f:
    f.write(csv_content)

# Ahora sí podemos leerlo como si fuera un archivo real en disco
df = pd.read_csv('produccion.csv')

print("¡Archivo cargado exitosamente!")
print(df.head())
```

## 2. Carga de Datos (`read_csv`)

La función más usada. Pandas intenta ser inteligente y detectar separadores (coma, punto y coma) y encabezados.

```python
import pandas as pd
import io

# 1. Crear 'datos_planta.csv' (separado por comas)
content_planta = """id,sensor,valor
1,S1,98.5
2,S2,102.1
3,S1,99.2"""
with open('datos_planta.csv', 'w') as f:
    f.write(content_planta)

# 2. Crear 'datos_latam.csv' (separado por punto y coma)
content_latam = """id;fecha;monto
101;2023-01-01;1500.50
102;2023-01-02;2300.20"""
with open('datos_latam.csv', 'w') as f:
    f.write(content_latam)

# --- AHORA EJECUTAMOS LA CARGA ---

# Cargar un archivo local estándar
df1 = pd.read_csv('datos_planta.csv')
print("--- Datos Planta (Comas) ---")
print(df1)

# Si el archivo usa punto y coma (común en Latinoamérica)
df2 = pd.read_csv('datos_latam.csv', sep=';')
print("\n--- Datos Latam (Punto y coma) ---")
print(df2)
```

## 3. Inspección Inicial (El "Rayos X")

Antes de analizar, debes saber qué tienes. Nunca asumas que los datos están limpios.

### 3.1 Ver los datos (`head`, `tail`)
Muestra las primeras o últimas filas. Fundamental para verificar que se cargó bien.

```python
df.head() # Muestra las primeras 5 filas por defecto
df.tail(3) # Muestra las últimas 3
```

### 3.2 Dimensiones (`shape`)
Devuelve una tupla `(filas, columnas)`.

```python
print(df.shape) 
# Salida: (1000, 15) -> 1000 observaciones, 15 variables
```

### 3.3 Metadatos y Tipos (`info`, `dtypes`)
Este es el comando más crítico. Muestra qué **tipo de dato** asignó Pandas a cada columna.

```python
df.info()
```

Verás una salida como:
- `float64`: Números decimales (e.g., Temperatura).
- `int64`: Números enteros (e.g., Cantidad).
- `object`: Texto o mixto (e.g., ID_Lote, Categoría). **Ojo aquí:** Si ves un número cargado como `object`, Pandas leyó "basura" (quizás una letra entre los números) y no podrás operarlo matemáticamente.

## Herramientas Python
Este laboratorio introduce las siguientes funciones esenciales que usarás en todo el curso:
