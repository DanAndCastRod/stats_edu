# Introducción
> "El promedio de profundidad del río es 1 metro, pero me ahogué cruzándolo".

El promedio (media) es la métrica más abusada en la ingeniería. Si tienes un proceso donde el 99% de las piezas tardan 1 minuto y una pieza tarda 100 minutos, el promedio te mentirá. Aquí aprendemos a usar la herramienta correcta para cada distribución.

<Callout type="warning">
**Concepto Clave:** En distribuciones con "colas largas" (como salarios o tiempos de reparación), la **Mediana** es mucho más representativa que el Promedio.
</Callout>

## Objetivos de Aprendizaje
- Calcular e interpretar Media, Mediana y Moda.
- Identificar cuándo el promedio es engañoso (Outliers).
- Visualizar la ubicación de estas medidas en una curva de distribución.

# 1. Las Tres M's

### 1.1 Media Aritmética ($\bar{x}$)
Es el centro de gravedad de los datos. Se calcula sumando todo y dividiendo por $n$.

$$
\bar{x} = \frac{\sum x_i}{n}
$$

*   **Pros:** Usa toda la información. Es base para la varianza.
*   **Contras:** Muy sensible a valores extremos (Outliers).

### 1.2 Mediana ($\tilde{x}$)
Es el valor que divide los datos en dos mitades iguales (50% arriba, 50% abajo).

*   **Pros:** Robusta a outliers. Ideal para tiempos de ciclo, salarios, precios de vivienda.
*   **Contras:** Matemáticamente más difícil de manipular.

### 1.3 Moda ($Mo$)
Es el valor más frecuente (el pico de la montaña).

*   **Uso:** Vital en variables cualitativas o para detectar procesos multimodales (e.g., dos turnos de trabajo con rendimientos diferentes).

# 3. Cálculo con Python

Pandas hace esto trivial. La habilidad no es calcularlo, es saber qué función llamar.

```python
import pandas as pd

# Datos de tiempos de ciclo (en segundos)
df = pd.DataFrame({'tiempos': [45, 42, 44, 43, 45, 46, 300]}) # ¡300 es un outlier!

media = df['tiempos'].mean()   # 80.7s (Engañoso, parece lento)
mediana = df['tiempos'].median() # 45.0s (Realista)
moda = df['tiempos'].mode()[0]   # 45.0s

print(f"Media: {media:.1f}, Mediana: {mediana:.1f}")
```

<Callout type="note">
**Regla de Dedo:** Si |Media - Mediana| es muy grande, investiga outliers. Tu proceso puede estar fuera de control.
</Callout>
